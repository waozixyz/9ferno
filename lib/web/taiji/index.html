What is Taiji OS?

TaijiOS is a modern fork of Inferno® Distributed Operating System (originally developed at Bell Labs). It brings powerful distributed computing capabilities to modern systems with updated build tools and 9front driver integration.

Features

  • Limbo VM: Portable Dis bytecode runs on native hardware or hosted (Linux/BSD/macOS).
  • 9P: Every resource (CPU, file, device) is shared via the Styx protocol.
  • Modern Development: Leveraging our custom transpiler Kryon you can easily make and port apps in a familiar qml like syntax.
  • UXN integration: Varvara virtual machine is integrated on the kernel level allowing you to emulate 8-bit uxntal applications natively on inferno.
  • Customizability: Bringing back simple windows 2000 era customization.

Quick start (NixOS)

        
# Clone the repository
git clone https://github.com/KryonLabs/TaijiOS.git
cd TaijiOS

# Enter the build shell
nix-shell

# Build and materialize the WM
./run-wm.sh
